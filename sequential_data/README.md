Requirements to run
---

To run the code, several requirements need to be met. All used code is written in Python3 and use the following libraries: 

- Pandas
- Numpy
- Scikit learn
- Matplotlib
- Seaborn
- Statsmodels
- Plotly

All libraries can be install using ```pip3```.
